# Simple Note Keeper

This project contains a LightningJS frontend that runs on a Vite dev server (port 3000) inside the `note_taking_frontend` directory.

Quick start:
- cd note_taking_frontend
- npm install
- npm run dev

Then open the provided preview URL for port 3000.

Features (frontend):
- Ocean Professional theme
- Notes list, editor, localStorage persistence
- Keyboard shortcuts: Cmd/Ctrl+N (new), Cmd/Ctrl+S (save)
- Responsive layout and subtle transitions
